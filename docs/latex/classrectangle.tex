\doxysubsection{Detailed Description}
Class to show a single Rectangle. 

A iso-\/rectangle can be represented as 2 x coordinates and 2 y coordinates or as 2 intervals (X interval and y interval) A iso-\/rectangle object can be created by any of the two inputs
