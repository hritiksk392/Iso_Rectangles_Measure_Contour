Function to generate stripes for Measure problem. 

Creates horizontal stripes in time O(nlogn) time using divide n conquer approach by dividing given set of Verical edges into 2 with equal size. ~\newline
 BASE CASE \+:= Given vector of Vertical Edge has only 1 element then create Stripe for single Edge ~\newline
 DIVIDE \+: Divide the vector of vertical edges into 2 by selecting median xm ~\newline
 CONQUER \+: Solve seperately for 2 sets of Vertical edges ~\newline
 MERGE \+: Merge the Stripe by copying, blackening and concatenating the two different Stripes

{\footnotesize\ttfamily long long measure (\begin{DoxyParamCaption}\item[{vector$<$ \textbf{ stripe} $>$}]{S }\end{DoxyParamCaption})}



To find Measure of n iso rectangle. 

In a given stripe the x union contains x\+\_\+interval of area intersected by rectangles. ~\newline
 In each stripe the width = y\+\_\+interval and summation of all disjoint x\+\_\+interval = xunion ~\newline
\begin{DoxyNote}{Note}
\+: Area covered by each stripe = (y\+\_\+interval) $\ast$ (x\+\_\+union) and hence sum for all to get Union area or final measure 
\end{DoxyNote}
