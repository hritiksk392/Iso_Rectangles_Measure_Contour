Creates horizontal stripes in time O(nlogn) time using divide n conquer approach by dividing given set of Verical edges into 2 with equal size. 

~\newline
BASE CASE \+:= Given vector of Vertical Edge has only 1 element then create Stripe for single Edge ~\newline
DIVIDE \+: Divide the vector of vertical edges into 2 by selecting median xm ~\newline
CONQUER \+: Solve seperately for 2 sets of Vertical edges ~\newline
MERGE \+: Merge the Stripe by copying, blackening and concatenating the two different Stripes
