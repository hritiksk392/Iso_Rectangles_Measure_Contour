Stripe for Contour Problem. 

Stripe for Measure Problem.

For contour problem X\+\_\+union will be a disjoint set of x intervals represented by a Binary search tree which contains interval on its leaves

X\+\_\+union will be a real number the size of each x\+\_\+union interval for measure problem
