void blacken ( vector< stripeC > &  S,
list< pair< interval, int >>  J

If For Sleft There exists a Right edge which does no have it's partner in Sleft then it's partner must be in SRight.

Hence the Right edge's xunion = x_interval
Similarly if Sright has a Left edge which does not have it's partner, then that stripeC must be blackened
Blacken function given a certain list of intervals will blacken those stripes which contain them

Parameters
S: Vector of stripeC to be applied
J: list of intervals to blackened with augemented Unique Id
Returns
blackens the S stripeC appropriately
Note
As the interval J is sorted and vector of stripeC is also sorted with Y coordinate it can be done in O(n) time using two pointer techniques

Concatenate two vector of Stripes
As both stripeC will have same partition of Y coordinates and sorted from CPY() method it can be done in 1 loop for all partitions
If one of the stripeC contain null x_union then Sfinal[i]'s x_union = x_union of non NUll stripeC
IF both are non null we have to create a new node with side = undefined and x =sleft.x_interval.top and sleft as Left child and sright as right child
This ensures the resultant tree is also binary search tree.

Finds the free interval for edge h from stripeC x_union.

Gets interval lying within h from S.x_union.
AS free interval will be complement of h wrt to h intersection S.x_union. So [x1,x2] and h intersect J = {[p1,p2],[p3,p4] ..}
Our free interval will be { [x1,p1], [p2,p3], [p4, p5],....[pn,x2]} where pi != pj

Creates horizontal stripes in time O(nlogn) time using divide n conquer approach by dividing given set of Verical edges into 2 with equal size.


BASE CASE := Given vector of Vertical Edge has only 1 element then create stripeC for single Edge
DIVIDE : Divide the vector of vertical edges into 2 by selecting median xm
CONQUER : Solve seperately for 2 sets of Vertical edges
MERGE : Merge the stripeC by copying, blackening and concatenating the two different Stripes

Parameters
V: Vector of Vertical Edges
x_extx interval of stripes
L: Contains the y-projections of all left edges whose partner is not in V.
R: Contains the y-projections of all right edges whose partner is not in V.
P: contains the y-projection of all endpoints of line segments in V plus the flame boundaries in y-direction, namely -inf, +inf
SOutput vector of Stripes
