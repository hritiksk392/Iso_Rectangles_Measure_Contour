void blacken ( vector< stripeM > &  S,
list< pair< interval, int >>  J

If For Sleft There exists a Right edge which does no have it's partner in Sleft then it's partner must be in SRight.

Hence the Right edge's xunion = x_interval
Similarly if Sright has a Left edge which does not have it's partner, then that stripeM must be blackened
Blacken function given a certain list of intervals will blacken those stripes which contain them

Parameters
S: Vector of stripeM to be applied
J: list of intervals to blackened with augemented Unique Id
Returns
blackens the S stripeM appropriately
Note
As the interval J is sorted and vector of stripeM is also sorted with Y coordinate it can be done in O(n) time using two pointer techniques

Function to generate stripes for Measure problem.

Creates horizontal stripes in time O(nlogn) time using divide n conquer approach by dividing given set of Verical edges into 2 with equal size.
BASE CASE := Given vector of Vertical Edge has only 1 element then create stripeM for single Edge
DIVIDE : Divide the vector of vertical edges into 2 by selecting median xm
CONQUER : Solve seperately for 2 sets of Vertical edges
MERGE : Merge the stripeM by copying, blackening and concatenating the two different Stripes

Parameters
V: Vector of Vertical Edges
x_extx interval of stripes
L: Contains the y-projections of all left edges whose partner is not in V.
R: Contains the y-projections of all right edges whose partner is not in V.
P: contains the y-projection of all endpoints of line segments in V plus the flame boundaries in y-direction, namely -inf, +inf
SOutput vector of Stripes

◆ measure()

long long measure ( vector< stripeM >  S)

To find Measure of n iso rectangle.

In a given stripeM the x union contains x_interval of area intersected by rectangles.
In each stripeM the width = y_interval and summation of all disjoint x_interval = xunion

Note
: Area covered by each stripeM = (y_interval) * (x_union) and hence sum for all to get Union area or final measure
